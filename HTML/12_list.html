List in HTML:

To decorate html document, we use list. html list are two types:

  1. Orderd list
  2. Unorderd list

Orderd List:

Start with Number:1

  1. PHP
  2. MySQL
  3. JavaScript

Start with choosen Number:5

  1. PHP
  2. MySQL
  3. JavaScript

Start with Letter:A

  1. PHP
  2. MySQL
  3. JavaScript

Start with Letter:a

  1. PHP
  2. MySQL
  3. JavaScript

Start with Roman:I

  1. PHP
  2. MySQL
  3. JavaScript

Start with Roman:i

  1. PHP
  2. MySQL
  3. JavaScript

Unorderd List:

Default-Radio

  • PHP
  • MySQL
  • JavaScript

Square

  • PHP

  • JavaScript
